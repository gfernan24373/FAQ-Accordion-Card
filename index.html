Es el lenguaje (si como lees, lenguaje ¬¬) de estilos de la web estandarizado por la W3C, utilizado por los desarrolladores para diseñar de manera más elegante y atractiva el contenido que es creado a través del documento html, es pocas palabras la presentación de la web.

El fundamento principal de CSS tiene que ver con la parte visual. Dado que los humanos somos seres visuales por naturaleza, aprender a usar CSS en tus proyectos puede lograr que tu web sea atractiva y amigable de ver (sin dejar de un lado la funcionalidad).

Existen dos formas de añadir CSS a tu archivo html, la primera agregando la etiqueta style dentro de la etiqueta head en tu archivo html para usar la sintaxis de css o bien puedes usar lo que se llama inline style que es agregando sintaxis css directamente a tus elementos o etiquetas, o utilizando un archivo externo con extensión .css en donde coloques todos los estilos que quieras usar. Está última es la opción más recomendada pero todo va a depender de la complejidad y el tamaño de tu proyecto.

Sí, puedes comenzar con un libro llamado "HTML & CSS" de Jhon Duckett que tiene un contenido muy visual y explícito para entender los fundamentos básicos sobre este interesante lenguaje.

Existen varios frameworks que han tenido una gran popularidad por su forma de "ahorrar código" y su "fácil entendimiento", entre ellos está bootstrap, bulma y tailwind.
